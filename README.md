# Live Pet Battle Simulator

A fun, real-time pet battle game where users can select, train, and battle virtual pets.

## Features
- Pet selection and retrieval by ID
- Real-time battles with WebSockets
- Mini-games for training
- Leaderboard

## Setup
1. Clone the repo
2. Install dependencies: `npm install` in backend and frontend
3. Start MongoDB
4. Run seed: `node database/seed.js`
5. Start backend: `npm start` in backend
6. Start frontend: `npm start` in frontend

## Hosting
- Backend: Deploy to Heroku
- Frontend: Deploy to Netlify or Vercel
- Database: Use MongoDB Atlas

## Tech Stack
- Frontend: React
- Backend: Node.js, Express, Socket.io
- Database: MongoDB
